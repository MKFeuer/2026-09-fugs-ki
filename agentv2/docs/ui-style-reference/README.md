# UI Style Reference

## Thema
Visuelle Orientierung an `agent/`.

## Beobachtungen
- Helle, warme Farbwelt mit viel Luft
- Google Sans Flex als klare, freundliche Schrift
- Zentrale Content-Breite statt voller Kante-zu-Kante-Fläche
- Runde Karten, weiche Borders und subtile Shadows
- Ruhige, leichte Animationen statt harter Übergänge
- Header oben, Chat im Mittelpunkt des Flows

## Wunsch für agentv2
- Das neue Layout soll den gleichen aufgeräumten Charakter behalten
- Die Oberfläche darf funktionaler werden, aber nicht visuell schwerer wirken
- Lagekarten, Canvas-Elemente und Tool-Events sollen in dasselbe ruhige Design passen

## Darkmode-Leitlinie
Wenn `agentv2` einen Darkmode bekommt, soll er denselben Charakter behalten:

- keine harten Schwarzflächen
- keine kalte Neonoptik
- dunkle, warme und ruhige Flächen
- klare Hierarchie zwischen App-Hintergrund, Panels und Karten
- Statusfarben weiter deutlich, aber nicht aggressiv

Der Darkmode soll wie eine dunkle Variante derselben Anwendung wirken, nicht wie ein anderes Produkt.
